# awesome-website
매우 주관적으로 선정한 멋진 웹사이트입니다.

### 선정 기준
- UI/UX : 멋진 외관과 사용자 경험 둘다 만족시키는 사이트를 선호합니다.
- 속도 : 보기에 멋지더라도 속도가 느리다면 사이트를 다 보기도 전에 꺼버리고 싶습니다.
- 감각 : 느낌적인 느낌이 있는지 개인의 주관이 매우 강하게 담겨있습니다.
